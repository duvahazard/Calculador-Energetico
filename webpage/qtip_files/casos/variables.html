Variables

Variable son los valores de orientación de los sistemas fotovoltaicos.

El altura es el ángulo entre el horizontal y donde esto orientado el fotovoltaico; 0 es una dispositivo vertical como una ventana en la casa y 90 grados es una dispositivo horizontal como una tragaluz en la casa.

El acimut es el orientación horizontal de la fotovoltiaco, una acimut de 0 es orientado el norte, 90 es orientado el este, 180 grados es orientado el sur (el mas recomendado en el hemisférica norte del planeta), y 270 es orientado el oeste.
